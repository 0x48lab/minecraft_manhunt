ゲームルール

🎯

基本ルール

ランナーはエンダードラゴンを倒すことが目標。ハンターはランナーを全員倒すことが目標です。

👥

チーム編成

最小2人から開始可能。ハンターとランナーが最低1人ずつ必要です。

🏁

勝利条件

  • ランナー勝利: エンダードラゴンを倒す
  • ハンター勝利: ランナーを全員倒す
💀

リスポーン

  • ハンター: 即座にリスポン
  • ランナー: 300秒後にリスポン（設定可能）
🌍

スポーン配置

ゲーム開始時、プレイヤーは動的な範囲内にランダム配置されます。

  • 2人: 500m範囲、20人: 2000m範囲
  • 20人以上はさらに拡大

ゲームモード

⏱️

時間制限モード

デフォルト40分の制限時間でプレイ

  • 時間切れで優勢度により勝敗決定
  • キルで所属チームに5分追加
  • リアルタイム優勢度表示
  • ランナーも即時リスポン
♾️

エンドレスモード

時間制限なしの従来型ゲームプレイ

  • 勝利条件達成まで継続
  • ランナーは300秒リスポン待機
  • 戦略的プレイ重視
  • 時間圧力なし

ゲームの流れ

1

準備フェーズ

参加者が役割を選択。最小人数が揃うとゲーム開始。

2

開始

プレイヤーがランダムな位置にスポーン。ハンターにコンパスが配布される。

3

追跡

ハンターはコンパスを使ってランナーを追跡。ランナーは装備を整えながら逃走。

4

決着

どちらかの勝利条件が満たされるとゲーム終了。MVPが発表される。

主要機能

🧭 追跡システム

仮想コンパス

物理コンパスまたは空手で右クリックして使用。ターゲットの方向と距離を表示。

ターゲット切り替え

複数のランナーを順番に追跡可能。異次元間でも機能。

⚠️ 警告システム

近接警告

ランナーにハンターの接近を3段階で警告（1/2/3チャンク）。

スコアボード表示

リアルタイムで危険度を確認可能。

💰 経済システム

通貨獲得

アクションに応じてコインを獲得。ハンターはダメージ・キル報酬、ランナーは生存・進歩報酬。

ショップ

6カテゴリー40種類以上のアイテムを購入可能。役割別の制限あり。

🤝 チーム機能

バディシステム

チームメイトとペアを組んで相対座標を共有。

チームチャット

/rコマンドで味方だけにメッセージ送信。

名前表示システム

スプリント中（スニーク解除時）は名前が表示され、敵が3チャンク以内にいると移動でG獲得。

🎮 ゲーム管理機能

実績リセット

ゲーム開始時に全プレイヤーの実績を自動リセット。公平なスタートを実現。

自動夜スキップ

夜になると自動的に夜明け2分前まで時間を進める。モンスター出現時間を確保。

攻略のコツ

初動が重要

ゲーム開始直後は食料と基本ツールの確保を優先。木材3個と牛肉3個が初期配布されます。

ネザー突入のタイミング

ネザー到達で200G獲得。早めに突入して装備を整えるのも戦略の一つ。

ダイヤモンド収集

1個につき25G獲得。効率的に集めてショップで強力な装備を購入しよう。

近接警告の活用

警告レベルに応じて行動を変える。赤色警告なら即座に逃走、黄色なら警戒しながら作業継続。

バディーシステム

信頼できる仲間とバディーを組んで位置情報を共有。分散行動時も連携可能。

要塞の探索

要塞発見で300G獲得。エンダーアイを効率的に使って早期発見を目指そう。

コンパスの活用

右クリックでターゲット切り替え。最も近いランナーを優先的に追跡しよう。

ダメージ報酬

1ダメージにつき5G獲得。弓での遠距離攻撃も有効な資金源。

チーム連携

複数ハンターで挟み撃ち。/rコマンドで位置情報を共有しながら追い込もう。

即座リスポンの利点

死亡してもすぐに復活。アグレッシブな戦法で圧力をかけ続けよう。

ネザー封鎖

ポータル周辺を監視してランナーの進行を妨害。トラップ設置も効果的。

時間ボーナス活用

30秒ごとに1.5G獲得。長期戦でも資金が増え続けるので装備更新を忘れずに。

ショップの活用

状況に応じて適切なアイテムを購入。エンダーパールやポーションは緊急時の切り札。

スポーン配置

プレイヤー数に応じて配置範囲が変動。大人数では最大半径が拡大される。

MVP報酬

ゲーム終了時に最も活躍したプレイヤーが選出。積極的なプレイが評価される。

言語設定

日本語と英語に対応。個人設定で好みの言語を選択可能。

観戦モード

死亡中や観戦者はゲーム全体を俯瞰可能。情報収集に活用しよう。

設定カスタマイズ

config.ymlで細かい調整が可能。サーバーに合わせて最適化しよう。

ダウンロード

必要環境

  • Minecraft 1.21.4
  • Spigot / Paper サーバー
  • Java 21以上
  • 最小プレイヤー数: 2人

インストール方法

  1. Modrinthから最新版をダウンロード
  2. JARファイルをpluginsフォルダに配置
  3. サーバーを再起動
  4. config.ymlで設定をカスタマイズ
